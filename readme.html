Funcionalidades Implementadas:
Criação de uma Base de Dados tendo em conta os requisitos do projeto
Criação de um WebServices Json em PHP que permite a pesquisa de notícias pela Data de início e fim e tags
Listagem das últimas notícias
Criação de um formulário para inserir uma notícia

Notas:
A base de dados foi desenhada de forma a que se possa adicionar acções nos diversos tipos de permissões. Para testar por exemplo, a funcionalidade pedida nesta primeira entrega use os logins que constam em install/exampleData.sql, onde podem encontrar as password de cada utilizador nos comentários. Foi implementado o padrão MVC de modo a isolar o acesso à base de Dados com a class DataController que consta em datacontroller.php. Foi usado um editor de html online "tinyeditor" que é open-source.
